# BalancingAct

Project website for **BalancingAct: Waypoint Guided Reinforcement Learning For Dynamically Sensitive Tasks**.

## Local preview

```bash
python3 -m http.server 8080
```

Then open [http://localhost:8080](http://localhost:8080).

## Structure

- `index.html`: project page
- `static/papers/balancing_act.pdf`: paper PDF
- `static/images/`: figures and task screenshots
- `static/videos/`: LQR vs. BalancingAct comparison rollouts
- `media/`: original media assets

## Deployment

This repo is set up for [GitHub Pages](https://pages.github.com/). Enable Pages from the `main` branch root to publish.
